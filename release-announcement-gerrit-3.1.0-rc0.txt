Gerrit version 3.1.0-rc0 is now available. This is the first release candidate
for 3.1.

We plan to make the final release during the upcoming hackathon (11-15 November),
and will make further release candidates in the coming weeks depending on volume
of fixes submitted.

Please report any issues on the tracker using the label "Blocking-3.1":
https://bugs.chromium.org/p/gerrit/issues/list?q=label%3ABlocking-3.1

Release notes are sparse at the moment, but will be updated before the final
release.
