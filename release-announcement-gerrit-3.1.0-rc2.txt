Gerrit version 3.1.0-rc2 is now available. This is the third release candidate
for 3.1.0 including several fixes and improvements since rc1.

There will be one more release candidate at the end of next week, and then the
final release will be done during the upcoming hackathon in Sunnyvale (11-15 Nov).

Please report any issues on the tracker using the label "Blocking-3.1":
https://bugs.chromium.org/p/gerrit/issues/list?q=label%3ABlocking-3.1
